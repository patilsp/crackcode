## Top 40 HTML Interview Questions and Answers

### 1. What is HTML?
HTML (HyperText Markup Language) is the standard markup language used to create web pages and web applications. It provides the structure and content of a webpage.

### 2. What is the difference between HTML and XHTML?
- HTML is more forgiving with syntax errors
- XHTML requires strict XML syntax
- XHTML elements must be properly nested and closed
- XHTML is case-sensitive

### 3. What are Semantic HTML elements?
Semantic elements clearly describe their meaning to both browser and developer:
```html
<header> - Defines a header for a document
<nav> - Defines navigation links
<main> - Specifies the main content
<article> - Defines an article
<section> - Defines a section
<aside> - Defines content aside from the content
<footer> - Defines a footer for a document
```

### 4. What is the difference between `<div>` and `<span>`?
- `<div>` is a block-level element
- `<span>` is an inline element
- `<div>` creates a new line before and after
- `<span>` only takes up as much width as necessary

### 5. What are HTML5 APIs?
HTML5 introduced several new APIs:
- Geolocation API
- Drag and Drop API
- Web Storage API
- Web Workers API
- WebSocket API
- Canvas API

### 6. What is the purpose of the `<!DOCTYPE>` declaration?
- Specifies the HTML version
- Helps browser render the page correctly
- Must be the first line in HTML document
- Different versions have different declarations

### 7. What is the difference between `localStorage` and `sessionStorage`?
- `localStorage` persists data until explicitly deleted
- `sessionStorage` persists data only for the session
- `localStorage` has no expiration time
- `sessionStorage` is cleared when the page session ends

### 8. What are HTML Forms and their elements?
```html
<form>
  <input type="text"> - Text input
  <input type="password"> - Password input
  <input type="radio"> - Radio buttons
  <input type="checkbox"> - Checkboxes
  <select> - Dropdown list
  <textarea> - Multi-line text input
  <button> - Submit button
</form>
```

### 9. What is the difference between GET and POST methods?
- GET: Data is visible in URL
- POST: Data is not visible in URL
- GET: Limited data size
- POST: No size limitations
- GET: Not secure for sensitive data
- POST: More secure for sensitive data

### 10. What is the purpose of the `alt` attribute in images?
- Provides alternative text for images
- Helps with accessibility
- Shows when image fails to load
- Important for SEO

### 11. What is the difference between `id` and `class`?
- `id` must be unique in the document
- `class` can be used multiple times
- `id` has higher specificity in CSS
- `class` is more reusable

### 12. What is the purpose of the `meta` tag?
```html
<meta charset="UTF-8"> - Character encoding
<meta name="viewport" content="width=device-width"> - Responsive design
<meta name="description" content="..."> - Page description
<meta name="keywords" content="..."> - Keywords for SEO
```

### 13. What is the difference between `async` and `defer` in script tags?
- `async`: Script loads and executes asynchronously
- `defer`: Script loads asynchronously but executes after HTML parsing
- `async` doesn't guarantee execution order
- `defer` maintains execution order

### 14. What is the purpose of the `canvas` element?
- Used for drawing graphics via JavaScript
- Provides a drawing surface
- Can be used for animations
- Supports pixel manipulation

### 15. What is the difference between `article` and `section`?
- `article`: Self-contained composition
- `section`: Thematic grouping of content
- `article` can contain `section`
- `section` can contain `article`

### 16. What is the purpose of the `figure` and `figcaption` elements?
```html
<figure>
  <img src="image.jpg" alt="Description">
  <figcaption>Caption for the image</figcaption>
</figure>
```

### 17. What is the difference between `strong` and `b` tags?
- `strong` indicates strong importance
- `b` is purely visual bold text
- `strong` has semantic meaning
- `b` is presentational only

### 18. What is the purpose of the `data-*` attributes?
- Custom data attributes
- Can store extra information
- Accessible via JavaScript
- No predefined meaning

### 19. What is the difference between `em` and `i` tags?
- `em` indicates emphasis
- `i` is purely visual italic text
- `em` has semantic meaning
- `i` is presentational only

### 20. What is the purpose of the `details` and `summary` elements?
```html
<details>
  <summary>Click to expand</summary>
  <p>Hidden content here</p>
</details>
```

### 21. What is the difference between `picture` and `img` elements?
The `picture` element provides more flexibility for responsive images:
```html
<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="Responsive image">
</picture>
```
Benefits:
- Art direction control
- Multiple image sources
- Better performance
- Responsive images

### 22. What is the purpose of the `template` element?
The `template` element holds HTML that won't be rendered immediately:
```html
<template id="user-card">
  <div class="card">
    <h2></h2>
    <p></p>
  </div>
</template>

<script>
  const template = document.querySelector('#user-card');
  const card = template.content.cloneNode(true);
  // Modify and append to DOM
</script>
```

### 23. What is the difference between `contenteditable` and `designMode`?
- `contenteditable`: Makes specific elements editable
- `designMode`: Makes entire document editable
```html
<div contenteditable="true">Editable content</div>
```

### 24. What is the purpose of the `slot` element in Web Components?
The `slot` element is used in Shadow DOM to create placeholder content:
```html
<template id="user-profile">
  <div>
    <slot name="name">Default Name</slot>
    <slot name="email">Default Email</slot>
  </div>
</template>
```

### 25. What is the difference between `preload` and `prefetch`?
```html
<link rel="preload" href="critical.js" as="script">
<link rel="prefetch" href="next-page.js">
```
- `preload`: High priority, current page
- `prefetch`: Low priority, future pages

### 26. What is the purpose of the `dialog` element?
The `dialog` element creates modal dialogs:
```html
<dialog>
  <h2>Modal Title</h2>
  <p>Modal content</p>
  <button onclick="this.closest('dialog').close()">Close</button>
</dialog>
```

### 27. What is the difference between `aria-label` and `aria-labelledby`?
- `aria-label`: Direct text description
- `aria-labelledby`: References another element's ID
```html
<button aria-label="Close dialog">X</button>
<div aria-labelledby="title">Content</div>
<h2 id="title">Section Title</h2>
```

### 28. What is the purpose of the `search` input type?
The `search` input type provides specialized search functionality:
```html
<input type="search" 
       placeholder="Search..." 
       results="5" 
       autosave="search-history">
```

### 29. What is the difference between `defer` and `module` type in script tags?
```html
<script defer src="script.js"></script>
<script type="module" src="module.js"></script>
```
- `defer`: Delays execution
- `module`: Enables ES6 module features

### 30. What is the purpose of the `download` attribute?
The `download` attribute forces download instead of navigation:
```html
<a href="file.pdf" download="custom-filename.pdf">Download PDF</a>
```

### 31. What is the difference between `hidden` and `display: none`?
- `hidden`: HTML attribute, semantic hiding
- `display: none`: CSS property, visual hiding
```html
<div hidden>Not visible</div>
<div style="display: none">Also not visible</div>
```

### 32. What is the purpose of the `spellcheck` attribute?
The `spellcheck` attribute enables browser spell checking:
```html
<textarea spellcheck="true">Check spelling</textarea>
```

### 33. What is the difference between `autocomplete` and `autofill`?
- `autocomplete`: Browser's form filling feature
- `autofill`: Browser's automatic form completion
```html
<input type="text" autocomplete="name">
```

### 34. What is the purpose of the `sandbox` attribute in iframes?
The `sandbox` attribute restricts iframe capabilities:
```html
<iframe src="page.html" sandbox="allow-scripts allow-forms"></iframe>
```

### 35. What is the difference between `required` and `aria-required`?
- `required`: Form validation
- `aria-required`: Accessibility information
```html
<input type="text" required aria-required="true">
```

### 36. What is the purpose of the `datetime` input type?
The `datetime` input type provides date and time selection:
```html
<input type="datetime-local" 
       min="2024-01-01T00:00" 
       max="2024-12-31T23:59">
```

### 37. What is the difference between `formaction` and `form` attributes?
- `formaction`: Overrides form's action
- `form`: Associates input with form
```html
<form id="myForm">
  <input type="submit" formaction="/custom-action">
</form>
<input type="text" form="myForm">
```

### 38. What is the purpose of the `translate` attribute?
The `translate` attribute controls translation behavior:
```html
<p translate="no">Don't translate this</p>
<p translate="yes">Translate this</p>
```

### 39. What is the difference between `pattern` and `title` attributes?
- `pattern`: Regular expression validation
- `title`: Tooltip and validation message
```html
<input type="text" 
       pattern="[A-Za-z]{3}" 
       title="Three letters only">
```

### 40. What is the purpose of the `enterkeyhint` attribute?
The `enterkeyhint` attribute suggests enter key behavior:
```html
<input type="text" enterkeyhint="search">
```

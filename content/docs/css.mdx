## ðŸ“š Key CSS Topics to Study  

### 1. **Selectors & Specificity**  
- How to target HTML elements (classes, IDs, pseudo-classes).  
- Specificity hierarchy (`!important`, inline styles, IDs > classes > elements).  

### 2. **Box Model**  
- `width`, `height`, `padding`, `border`, `margin`.  
- `box-sizing: border-box` vs `content-box`.  

### 3. **Layout Systems**  
- **Flexbox**: 1D layouts (align items, justify content).  
- **CSS Grid**: 2D layouts (rows, columns, gaps).  
- **Positioning**: `static`, `relative`, `absolute`, `fixed`, `sticky`.  

### 4. **Responsive Design**  
- Media queries (`@media`).  
- Relative units (`rem`, `vw`, `%`).  
- Mobile-first vs desktop-first approaches.  

### 5. **Transitions & Animations**  
- `transition` for simple effects.  
- `@keyframes` for complex animations.  

### 6. **CSS Architecture**  
- BEM naming convention.  
- CSS-in-JS (Styled Components, Emotion).
